Lightning Fast

No build step required. Just include the script and start using chai-* classes. Your styles are applied instantly at runtime.

🧩

Composable

Mix and match utilities freely. Combine chai-flex, chai-gap-4, and chai-p-6 to build any layout.

🔄

Dynamic

A MutationObserver watches for new elements. Add chai-* classes dynamically via JS and they're styled automatically.
